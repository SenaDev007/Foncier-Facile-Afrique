## 🎯 Caractéristiques des images

- **Dimensions optimales :** 400x300px minimum
- **Format :** JPG (compression web)
- **Qualité :** 85% (bon équilibre qualité/poids)
- **Résolution :** 72dpi (optimisé pour web)

## 🚀 Instructions

1. Créer le dossier `services` dans `public/images/`
2. Télécharger chaque image avec le nom exact spécifié
3. Placer les 4 fichiers dans le dossier
4. Rafraîchir la page pour voir les images animées
