# 🖼️ Solution Image Hero — Instructions Rapides

## 🚨 **Problème identifié**
L'image de background du hero ne s'affiche pas car le fichier `/public/images/hero/hero-bg.jpg` n'existe pas encore.

## 📋 **Solution immédiate (2 minutes)**

### 1. Télécharger l'image
**URL directe :**
```
https://images.unsplash.com/photo-1582407947304-fd86f028f716?w=1920&q=85
```

### 2. Enregistrer sous
**Nom du fichier :** `hero-bg.jpg`
**Destination :** `d:\Projet YEHI OR Tech\Site web- Foncier Facile Afrique\public\images\hero\`

### 3. Instructions détaillées
1. Ouvrir le lien dans le navigateur
2. Faire clic droit sur l'image → "Enregistrer l'image sous..."
3. Naviguer vers le dossier `public/images/hero/`
4. Enregistrer avec le nom exact `hero-bg.jpg`

## 🎯 **Résultat attendu**
- ✅ Image de fond immersive dans le hero
- ✅ Vue aérienne de maisons (contexte immobilier africain)
- ✅ Overlay dégradé préservé pour lisibilité du texte
- ✅ Effet parallax fonctionnel (une fois Framer Motion installé)
